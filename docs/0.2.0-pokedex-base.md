# Versión 0.2.0 - Base de la Pokédex

## Resumen
- Se habilitó el enrutamiento basado en `pages` para soportar `getStaticProps` y `getStaticPaths`.
- Se generó la lista estática de los 151 Pokémon de Kanto utilizando un componente reutilizable `PokemonCard`.
- Se implementó la página de detalles con información extendida (tipos, altura, peso, habilidades y estadísticas base).
- Se añadió un adaptador ligero compatible con Axios para mantener la consistencia en la obtención de datos a través de `fetch`.

## Instrucciones de Uso
1. Ejecuta `npm run dev` para iniciar el entorno de desarrollo.
2. Visita `http://localhost:3000` para navegar por la Pokédex.
3. Haz clic en cualquier carta para abrir la vista detallada del Pokémon seleccionado.
