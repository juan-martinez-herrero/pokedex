# Versión 0.3.0 - Selector de vista para la Pokédex

## Resumen
- Se añadió un selector en la página principal que permite cambiar entre el diseño tradicional de tarjetas y un nuevo modo en lista horizontal.
- Se creó el componente `PokemonListRow` para reutilizar la información de cada Pokémon en el nuevo modo de presentación.

## Detalles técnicos
- El estado local `viewMode` controla el modo activo y se comparte con el selector tipo botón-segmentado.
- En modo tarjetas se conserva la cuadrícula existente, mientras que el modo lista renderiza una columna con elementos flexibles.
- Ambos componentes (`PokemonCard` y `PokemonListRow`) comparten la utilidad `capitalize` definida en `lib/strings.ts` para mantener consistencia en los nombres.

## Próximos pasos sugeridos
- Recordar la selección de vista del usuario utilizando almacenamiento local o parámetros de la URL.
- Añadir atajos de teclado o gestos para alternar rápidamente entre los modos de visualización.
