# Versión 0.3.1 - Buscador por nombre en la Pokédex

## Resumen
- Se incorporó un campo de búsqueda en la página principal para filtrar Pokémon al escribir su nombre.
- Se añadió un mensaje de estado cuando no existen coincidencias para mejorar la retroalimentación del usuario.

## Detalles técnicos
- El estado local `searchTerm` almacena la consulta activa y se memoiza la lista filtrada para evitar cálculos innecesarios.
- Se reutiliza el listado original en modo tarjetas y lista, renderizando únicamente los elementos que coinciden con el texto ingresado.
- Se añadió un mensaje accesible que se muestra en ausencia de resultados para mantener informada a la persona usuaria.

## Próximos pasos sugeridos
- Permitir la búsqueda por identificador numérico o por coincidencias parciales en otros idiomas.
- Combinar la búsqueda con filtros por tipo o estadísticas para refinar aún más los resultados.
