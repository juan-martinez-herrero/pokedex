# Versión 0.2.1 - Configuración de imágenes remotas

## Resumen

Se ajustó la configuración de Next.js para permitir que `next/image` cargue las ilustraciones oficiales de los Pokémon hospeda
as en `raw.githubusercontent.com`.

## Detalles Técnicos

- Se añadió un `remotePattern` en `next.config.ts` que habilita las rutas de sprites oficiales de la PokeAPI.
- No se realizaron cambios en componentes o páginas.

## Pruebas

- `npm run build` *(falla en este entorno por falta de acceso de red a la PokeAPI durante la generación estática)*
